# kembangseladang-web

Website Publik Kembang Seladang.

## Deskripsi
Kembang Seladang website publik (kembangseladang.com) adalah platform e-commerce katalog/etalase bunga yang terintegrasi dengan WhatsApp checkout ke admin toko (Priska: 0877-7263-6627). Website ini juga berfungsi sebagai push SEO melalui fitur artikel/blog.

## Tech Stack
- Frontend: Vite + React 19 + TypeScript
- Styling: TailwindCSS v4
- State Management: Zustand (Cart) + TanStack React Query (Server cache)
- Database/BaaS: Supabase (Sharing database dengan admin panel)
